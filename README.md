# Project_3 FitneZone

### Overview

Created MERN (MongoDB/Express/React/Node.js) Stack application called FitneZONE.
Where user can create a profile, search for a receipe, workout instructions, and can post or view
posts from other users.

### Technology used:
    * HTML
    * CSS
    * JavaScript
    * Node.js
    * React
    * MongoDB
